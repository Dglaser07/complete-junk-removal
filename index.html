⚠ This data lives only on this device

There's no cloud account behind this app — everything you see is stored locally in this browser. That means it can be lost if: the browser's site data gets cleared, this phone is lost/damaged/replaced, storage gets corrupted, the app is reinstalled from scratch, or you move the app to a different web address. Exporting a backup regularly is the only real protection against any of that.

Export a backup

Downloads everything — settings, customers, jobs, archive, and photos — as one file. Use this to move data between devices when you're not logged into the same account, or just to keep a safe copy.
